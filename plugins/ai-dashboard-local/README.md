# ai-dashboard-local

Cross-platform local AI Dashboard workflow plugin for Claude Code.

## Included
- Commands: `/dashboard-start`, `/dashboard-refresh`, `/dashboard-status`, `/dashboard-stop`
- Skill: `dashboard-ops`

## What it does
- Starts/stops dashboard runtime
- Rebuilds icon config from roots
- Checks runtime health quickly
- Provides repeatable troubleshooting workflow

## Runtime dependency
This plugin expects dashboard assets in the same repository root (`ai-dashboard-refactor`).
